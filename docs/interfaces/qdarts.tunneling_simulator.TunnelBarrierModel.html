qdarts.tunneling_simulator.TunnelBarrierModel¶

class qdarts.tunneling_simulator.TunnelBarrierModel(gate_offsets, gate_levers=None)¶

Model of the tunnel barriers of a device

This class defines a mapping between gate voltages of the device and the tunnel coupling between the dots. To be more exact, the tunnel coupling between dots i, and j and the gate voltages v is given by

T_ij = exp(W_ij^Tv+b_ij)

where W_ij is a vector of couplings and b_ij is an offset.

__init__(gate_offsets, gate_levers=None)¶

Creates a tunnel barrier model.

Parameters:

  • gate_offsets (NxN np.array of floats) – the offsets b_ij provided as matrix.

  • gate_levers (NxNxK np.array of floats or None) – Here, K is the number of plunger gates. The first two indices describe the index of the tunnel coupling matrix ij. If None, it is assumed to be 0.

Takes an affine subspace of the simulated model.

Let v=Pv’+m. Computes a new parameterization such, that T_ij = exp(W’_ij^Tv’+b’_ij)
