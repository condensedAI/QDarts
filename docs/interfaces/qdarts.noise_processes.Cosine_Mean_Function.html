qdarts.noise_processes.Cosine_Mean_Function¶

class qdarts.noise_processes.Cosine_Mean_Function(noise_model, a, W, b=None)¶

Decorator of an random process that models an additive mean term that depends on the gate voltages. This term is added to noise values sampled from the decorated noise model

The mean term of the ith noise element is given as a set of cosine functions:

\(\mu_i(v)= \sum_j a_{ij} \cos(2\pi (w_{ij}^T v+b_{ij}))\)

and the resulting noise is

\(\epsilon(v)_{t}=\mu(v) + \epsilon(v)^D_{t}\)

where \(\epsilon(v)^D_{t}\) is the decorated noise process.

The user supplies the weight tensor W with elements \(W_{ijk}\) so that W[i,j] is the vector \(w_{ij}\) and a matrix a with the amplitude values \(a_{ij}\). Finally, b is the matrix of offsets \(b_{ij}\), which can be left as None, in which case it is sampled uniformly between 0 and 1.

__init__(noise_model, a, W, b=None)¶

Initialized the cosine mean function.

Parameters:

  • noise_model (AbstractNoiseProcess) – The noise process with N dimensions to decorrate.

  • a (NxM np.array of float) – Amplitudes of the M overlapping cosine functions

  • W (NxMxK np.array of float) – K dimensional weights of the M consine functions for the N outputs.

  • b (NxM np.array of float or None) – Phases of the M overlapping cosine functions. If none, it is drawn uniformly between 0 and 1.
