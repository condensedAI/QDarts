QDarts
======

Installation
------------
Installation can be done via ``python build``. In the QDarts directory, run::

    make build
    python3 -m pip install QDarts/dist/QDarts-<Version>-py3-none-any.whl
    
where ``<Version>`` is the current QDarts version.
